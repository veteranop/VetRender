Getting Started

What is VetRender?

VetRender is a professional RF propagation analysis tool designed for wireless engineers, broadcast technicians, and radio enthusiasts. It provides accurate signal coverage predictions using advanced propagation models and terrain analysis.

Key Features

  • RF Propagation Modeling - Calculate coverage areas with free-space path loss
  • Terrain Analysis - Account for terrain blockage and diffraction effects
  • Custom Antenna Patterns - Import real antenna patterns from XML files
  • AI-Powered Tools - Automatically extract antenna data from PDF datasheets
  • Interactive Maps - Multiple basemap options with zoom and pan
  • Project Management - Save and load complete project configurations

Quick Start

  1. Launch VetRender - Double-click the desktop icon or start from the Start Menu
  2. Set Your Location - Right-click on the map and select "Set Transmitter Location"
  3. Configure Transmitter - Right-click and select "Edit Transmitter Configuration"
    • Set ERP (Effective Radiated Power in dBm)
    • Set Frequency (MHz)
    • Set Antenna Height (meters above ground)
  4. Calculate Coverage - Click the "Calculate" button on the toolbar
  5. Analyze Results - View the coverage overlay and probe signal strength by right-clicking

Pro Tip

Use Ctrl+S frequently to save your project. VetRender automatically saves your last session, but manual saves preserve your entire configuration including plots.

RF Propagation Modeling

How It Works

VetRender calculates RF propagation using industry-standard free-space path loss models combined with optional terrain analysis. The calculation considers:

  • Transmitter Power (ERP) - Effective Radiated Power in dBm
  • Frequency - Operating frequency affects path loss
  • Distance - Path loss increases with distance
  • Antenna Pattern - Directional gain/loss in each direction
  • Terrain (Optional) - Blockage and diffraction effects

Calculation Process

  1. Grid Generation - Creates a polar grid around the transmitter
  2. Path Loss Calculation - Computes free-space path loss for each point
  3. Antenna Gain Application - Applies antenna pattern gain/loss per azimuth
  4. Terrain Analysis (if enabled) - Calculates terrain blockage and diffraction
  5. Received Power - Combines all factors to predict signal strength
  6. Visualization - Generates color-coded coverage overlay

Coverage Distance

Select coverage radius from Plot Settings > Coverage Distance:

  • 25 km - Local/urban coverage
  • 50 km - Regional coverage
  • 75-100 km - Standard broadcast range
  • 125-200 km - Long-range/rural coverage
  • Custom - Enter specific radius (1-500 km)

Signal Threshold

The signal threshold (default: -110 dBm) determines what's considered "coverage." Points above this threshold are shown in colored contours, points below are shown as no signal.

Typical Signal Thresholds

  • -70 dBm - Excellent signal (cell phones)
  • -85 dBm - Good signal (Wi-Fi)
  • -95 dBm - Fair signal (FM radio)
  • -110 dBm - Minimum usable (weak signal)

Antenna Patterns

Default Antenna

VetRender starts with a default omnidirectional antenna (0 dBi gain). This provides equal coverage in all directions.

Importing Antenna Patterns

Manual Import (XML Files)

  1. Right-click on the map
  2. Select "Edit Antenna Information"
  3. Click "Load Pattern from XML"
  4. Select your antenna pattern file (.xml format)

XML Format Requirements

Antenna pattern XML files should contain:

  • Azimuth angles (0-360¬∞)
  • Gain values (dBi) for each azimuth
  • Optional: Elevation patterns
  • Optional: Metadata (manufacturer, model, etc.)

Terrain Analysis

Enabling Terrain Mode

Check Plot Settings > Use Terrain Analysis to include terrain effects in calculations.

‚ö†Ô∏è Important

Terrain analysis requires internet connection to download elevation data. The first calculation may take longer as data is cached.

How Terrain Analysis Works

VetRender uses segment-by-segment terrain diffraction analysis:

  1. Elevation Data Retrieval - Downloads terrain elevation for your coverage area
  2. Line-of-Sight Analysis - Checks if terrain blocks signal path
  3. Fresnel Zone Calculation - Computes required clearance for each frequency
  4. Diffraction Loss - Calculates signal loss over terrain obstacles
  5. Shadow Zones - Identifies areas completely blocked by terrain

Shadow Zones

Enable shadow zone visualization: View > Layers > Shadow Zones

Shadow zones appear as dark areas where terrain completely blocks signal propagation. These areas typically receive no usable signal.

Terrain Cache

Downloaded elevation data is cached locally for faster subsequent calculations. Manage cache from the context menu: Manage Cache

Settings & Configuration

Transmitter Configuration

Access via right-click menu: Edit Transmitter Configuration

Parameters:

  • ERP (Effective Radiated Power)
    • Measured in dBm (decibels relative to 1 milliwatt)
    • Typical range: 10 dBm (10 mW) to 80 dBm (100 kW)
    • Higher values = greater coverage area
  • Frequency
    • Operating frequency in MHz
    • Affects path loss calculation
    • Higher frequencies have greater path loss
  • Antenna Height
    • Height above ground in meters
    • Greater height = better line-of-sight
    • Especially important for terrain analysis
  • Signal Threshold
    • Minimum signal strength for coverage (dBm)
    • Default: -110 dBm
    • Adjust based on receiver sensitivity

How the AI Works

VetRender uses a local AI model (Ollama) to analyze PDF documents:

  • Text Extraction - Reads text and tables from PDF
  • Pattern Recognition - Identifies azimuth/elevation data
  • Validation - Checks data consistency and plausibility

Troubleshooting

Common Issues

Map Won't Load

  • Check internet connection (maps require online access)
  • Try a different basemap from View > Basemap
  • Clear map cache: Right-click > Manage Cache > Clear Cache
  • Restart VetRender

Calculation Fails or Takes Too Long

  • Reduce coverage distance (try 50 km instead of 200 km)
  • Lower quality preset (use Medium instead of Ultra)
  • Disable terrain analysis for faster calculations
  • Reduce terrain detail level

Terrain Data Not Loading

  • Check internet connection (terrain data downloads from server)
  • Verify coverage area is on land (ocean areas have no terrain)
  • Try clearing terrain cache and recalculating
  • Some remote areas may have limited elevation data

Antenna Pattern Not Loading

  • Verify XML file format is correct
  • Check that azimuth values range from 0-360¬∞
  • Ensure gain values are in dBi
  • Try resetting to default omni antenna and reload

Coverage Looks Incorrect

  • Verify ERP is set correctly (common mistake: using Watts instead of dBm)
  • Check frequency setting
  • Verify antenna height is realistic
  • If using terrain, try disabling it to see free-space coverage
  • Check signal threshold setting

Performance Tips

  • Start with Medium quality, increase if needed
  • Cache maps before field use (work offline)
  • Use smaller coverage radii for faster calculations
  • Close other applications to free memory
  • Terrain analysis is much slower - disable for quick previews

RF Terminology

Term Definition
ERP Effective Radiated Power - Total power radiated by antenna
dBm Decibels relative to 1 milliwatt (power measurement)
dBi Decibels relative to isotropic antenna (gain measurement)
FSPL Free Space Path Loss - Signal loss due to distance
Azimuth Horizontal angle from north (0-360¬∞)
Fresnel Zone Ellipsoid area around line-of-sight path that affects signal
LOS Line of Sight - Direct path between transmitter and receiver
