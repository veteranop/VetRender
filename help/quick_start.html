Get started with RF propagation analysis in 5 minutes

Welcome to Cellfire RF Studio!

Cellfire RF Studio is your professional RF propagation analysis tool. This guide will get you analyzing coverage patterns in just a few steps.

Step 1: Set Your Transmitter Location

  1. Right-click anywhere on the map
  2. Select "Set Transmitter Location" from the context menu
  3. The transmitter marker (red) will appear at that location

Alternative: Use the toolbar location display to enter exact coordinates

Step 2: Configure Your Transmitter

  1. Right-click on the map again
  2. Select "Edit Transmitter Configuration"
  3. Set your parameters:
    • ERP: Effective Radiated Power (in dBm, e.g., 40 dBm = 10W)
    • Frequency: Operating frequency in MHz (e.g., 88.5 for FM)
    • Antenna Height: Height above ground in meters
  4. Click OK to save

Step 3: Calculate Coverage

  1. Click the "Calculate" button on the toolbar
  2. Watch the progress bar as Cellfire RF Studio computes your coverage
  3. The coverage overlay will appear on the map showing signal strength

Color Guide:

  • üü¢ Green = Strong signal
  • üü° Yellow = Medium signal
  • üü† Orange = Weak signal
  • üî¥ Red = Very weak signal

Step 4: Analyze Your Results

  1. Right-click anywhere on the map
  2. Select "Probe Signal Strength Here"
  3. View detailed information:
    • Signal strength in dBm
    • Distance from transmitter
    • Azimuth (bearing)
    • Signal quality assessment

Step 5: Save Your Project

  1. Press Ctrl+S or use Project > Save Project
  2. Choose a filename (e.g., "MyStation.vtr")
  3. Your entire configuration is saved for later use

Common Tasks

Change the Map Style

  1. Go to View > Basemap
  2. Choose from Satellite, Street Map, or Topographic

Enable Terrain Analysis

  1. Go to Plot Settings > Enable Terrain
  2. Check the box to enable
  3. Recalculate your coverage

Import a Custom Antenna

  1. Right-click on map > Edit Antenna Information
  2. Click Load Pattern from XML
  3. Select your antenna pattern file

Generate a Report

  1. Calculate your coverage first
  2. Go to Export > Generate Report
  3. Configure report options
  4. Click Generate
