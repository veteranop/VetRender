# Media Assets for Cellfire RF Studio

This directory contains branding and media assets for Cellfire RF Studio (formerly VetRender).

## Directory Structure

- `logos/` - Logo files in various formats (PNG, SVG, etc.)
- `icons/` - Application icons for different platforms
- `splash/` - Splash screen images
- `report_assets/` - Images and graphics for PDF reports

## Usage in Code

Assets from this directory are used in:
- PDF report generation (cover page, headers, footers)
- Application splash screen
- About dialog
- Export watermarks

## File Naming Convention

- Use lowercase with underscores: `cellfire_logo_main.png`
- Include size in filename for multiple sizes: `icon_256x256.png`
- Use descriptive names: `report_header.png`, `splash_screen.png`
